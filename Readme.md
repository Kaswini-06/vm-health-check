# VM Health Check Script

A comprehensive bash script that monitors virtual machine health by analyzing CPU, memory, disk space, system load, and network connectivity.

## Features

- ✓ CPU Usage Monitoring
- ✓ Memory Usage Tracking
- ✓ Disk Space Analysis
- ✓ System Load Verification
- ✓ Process Count Monitoring
- ✓ System Uptime Information
- ✓ Network Interface Status
- ✓ Color-coded output for easy interpretation
- ✓ Detailed health reports with `--explain` flag

## Usage

```bash
# Basic health check
./vm-health-check.sh

# Display detailed explanation
./vm-health-check.sh --explain

# Show help
./vm-health-check.sh --help
